Typography

Phi uses a typographic scale of 1.25 of the base font-size. Based on 1em or 16 pixels, that creates a 20px vertical baseline. That means a great, logical vertical rhythm for every design element is based on 20pxs.

General Typography

The default vertical rhythm is based off of multiples of 20px (1.25em of default).
